If you are reading this message, congrats! You have come a long way! Now its time to put you to the test!

I encourage all of you to make a pull request telling us all about your self! (If you prefer to keep your privacy, you can make something up)

In order to do that, you can make a pull request to this project by adding the neccessary code to place a foto, title and description, if you dont remember how to do that you can get some inspiration with the ones that are already done!

It is quite simple to make a pull request, in short words yout just need to:
  • Find this project in the github repo
  • Make a copy of your own
  • Edit the code in your copy
  • Press the pull request button and document!
Feel free to customize your entry but please keep in mind that yout pull-request will be revised before beeing inserted.
